# master-unibuc-2026

## Quick start guide

Prerequisited:
 * Python environment
 * Jupyter notebook

**Getting started with Virtualenv**

```bash
virtualenv -p `which python3` venv
source venv/bin/activate
pip install notebook
git clone git@github.com:tiberiu44/master-unibuc-2026.git 
cd master-unibuc-2026
python3 -m notebook
```
